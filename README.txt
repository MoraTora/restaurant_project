Для запуска проекта необходимо:

1. В PostgreSQL создать пустую базу данных

2. В файле .env изменить строку подключения к базе данных PostrgeSQL, 
а именно в строке DB_URL = 'postgresql://username:password@host:port/database' заменить: 
username - имя пользователя для получения доступа к базе данных.
password — пароль для получения доступа к базе данных.
host — расположение сервера базы данных.
port — порт для подключения.
database — название базы данных.

3. В папке с проектом выполнить команду в консоли команду pip install -r requirements.txt

4. Для применения миграций к базе данных в папке с проектом выполнить следующую команду: alembic upgrade head

5. Выполнить команду uvicorn main:app --reload для запуска приложения для тестирования


<----------------Описание функциональных файлов проекта---------------->

Файл main.py содержит основную логику взаимодействия по API

Файл models содержит модели проекта

Файл utils.py содержит функции для CRUD операций а так же функции для олучения экземпляра модели и словаря для ответа API
